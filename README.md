# IM Demo

这是一个更完整一点的纯 IM demo，仍然保持单文件服务和内存数据，但补了更接近真实聊天产品的核心体验。

## 已升级的功能

- 切换当前账号
- 左侧会话列表选择联系人
- 会话搜索
- 当前会话消息搜索
- 会话置顶
- 未读数
- 按最近消息排序
- 时间分组
- 已送达 / 已读状态
- 草稿记忆
- 表情快捷输入
- 引用回复
- 消息复制
- 本地删除单条消息
- 撤回自己的消息
- SSE 实时刷新
- 清空当前内存消息

## 运行

```bash
node server.js
```

打开 [http://localhost:3000](http://localhost:3000)

## 说明

- 所有消息和置顶状态只保存在内存里，服务重启后会恢复初始示例数据
- 草稿使用浏览器本地存储，按“账号 + 联系人”分别保存
- 删除单条消息只对当前浏览器视角生效，模拟客户端本地删除
- 撤回消息默认允许 2 分钟内操作
- 当前预置 4 个用户：Alice、Bob、Charlie、Diana
